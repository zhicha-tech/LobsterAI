# 高级权限设计参考

## 权限体系概览

飞书多维表格的权限分三层：**系统角色 → 数据表权限 → 行列权限**。

```
系统角色（所有者/管理员/编辑者/阅读者/自定义角色）
  └── 数据表权限（可管理/可编辑/仅可阅读/无权限）
       ├── 记录权限（行）：可新增/可删除/可编辑范围/可阅读范围
       └── 字段权限（列）：逐字段设置可阅读/可新增/可编辑
```

**注意**：权限不可通过 API 配置，搭建时输出权限设计方案供用户手动操作。

---

## 系统角色

| 角色 | 权限范围 | 可修改 |
|------|---------|--------|
| **所有者** | 最高权限，所有数据表可管理 | 不可修改 |
| **管理员** | 同所有者，所有数据表可管理 | 不可修改 |
| **编辑者** | 数据表级别可设：可管理/可编辑/仅可阅读/无权限 | 可修改 |
| **阅读者** | 数据表级别仅可设：仅可阅读/无权限 | 可修改 |
| **自定义角色** | 同编辑者，可灵活配置 | 可修改 |

---

## 数据表权限 → 行列权限矩阵

| 数据表权限 | 记录权限（行） | 字段权限（列） |
|-----------|--------------|--------------|
| **可管理** | 全部行：增删改查，不可修改 | 全部列：增删改查，不可修改 |
| **可编辑** | 可设：是否可新增/删除；可编辑范围（所有记录/本人相关/特定条件）；可阅读范围 | 可选：所有字段可编辑 / 自定义（逐字段设阅读/新增/编辑） |
| **仅可阅读** | 可设可阅读范围（所有记录/指定记录） | 可选：所有字段可阅读 / 指定字段 |
| **无权限** | 无法访问 | 无法访问 |

---

## 可编辑记录范围（三种模式）

当数据表权限为"可编辑"时，可编辑和删除的记录范围有三种：

### 1. 所有记录
所有记录均可编辑和删除。

### 2. 与成员本人相关的记录
- 成员本人创建的记录
- 指定人员字段包含成员本人的记录（如"负责人"字段）
- 可选择人员字段及其扩展字段（如直属上级），不支持创建人/修改人字段

### 3. 满足特定条件的记录
- 自定义筛选条件（最多 10 个条件）
- 支持的条件字段：单选、多选、人员、创建人、流程、数字（数字/进度/评分/货币）、公式、查找引用
- 条件组内支持"或"/"且"关系

**阅读范围**：当可编辑范围不是"所有记录"时，可另设可阅读范围（所有/本人相关/特定条件/同可编辑范围）。实际可阅读 = 可编辑范围 ∪ 可阅读范围。

## 字段权限详解

### 所有字段内容可编辑
- 可看到并编辑所有字段内容
- 不能新增/删除字段、更改字段类型/名称
- 单选/多选字段不可增删改选项

### 自定义权限
逐字段设置：

| 权限 | 说明 | 依赖 |
|------|------|------|
| 可阅读 | 可查看字段内容 | — |
| 可新增 | 新增记录时可填写此字段 | 需先开启"可新增记录" |
| 可编辑 | 可修改此字段内容 | 自动获得可阅读+可新增 |

**特殊字段限制**：
- 索引列（首字段）：始终可阅读，不可隐藏
- 系统字段（创建人/修改人/创建时间/更新时间）：始终可阅读，不可编辑
- 公式/查找引用/自动编号/人员扩展字段：不支持可新增/可编辑
- 新增/删除字段、更改字段类型：需**可管理**权限

### 选项权限
单选/多选字段可设"可增删改选项"，仅对有编辑权限的字段生效。已在无编辑权限的记录中使用的选项不可删除。

### 附件导出权限
附件字段可设"可导出附件"，需至少有可阅读权限。还受全局"下载多维表格"权限控制。

## 典型权限设计场景

### 场景 1：管理者只看仪表盘

> 需求：管理者只查看数据概览，不接触源数据。

```
角色：数据查看者（自定义角色）
├── 仪表盘：仅可阅读
└── 所有数据表：无权限
```

### 场景 2：员工可填写但不可改结构

> 需求：员工在表中填写数据，但不能新建表、改字段。

```
角色：数据填写者（自定义角色）
└── 数据表：可编辑
    ├── 记录权限：可新增，可删除（可选）
    └── 字段权限：所有字段内容可编辑
```

### 场景 3：只能操作自己的数据

> 需求：每人只看到和编辑自己创建的或分配给自己的记录。

```
角色：个人数据角色（自定义角色）
└── 数据表：可编辑
    ├── 记录权限：
    │   ├── 可新增：是
    │   ├── 可删除：是
    │   ├── 可编辑范围：与成员本人相关的记录
    │   │   ├── 成员本人创建的记录
    │   │   └── "负责人"字段包含成员本人的记录
    │   └── 可阅读范围：上述设置为可编辑的记录
    └── 字段权限：所有字段内容可编辑
```

### 场景 4：部门数据隔离

> 需求：销售部只能看本部门的客户数据。

```
角色：销售部（自定义角色）
└── 客户表：可编辑
    ├── 记录权限：
    │   ├── 可编辑范围：满足特定条件的记录
    │   │   └── 条件：[所属部门] = "销售部"
    │   └── 可阅读范围：同可编辑范围
    └── 字段权限：自定义（隐藏敏感字段如"利润率"）
```

## 给用户的权限建议模板

搭建完成后输出权限设计建议：

```
### 建议的权限配置

#### 角色 1：[角色名]
- 成员：[谁应该被分配到此角色]
- 数据表权限：
  | 数据表 | 权限 |
  |--------|------|
  | 表1    | 可编辑 |
  | 表2    | 仅可阅读 |
- 记录权限：[所有记录 / 本人相关 / 特定条件]
- 字段权限：[所有可编辑 / 自定义（列出隐藏/只读字段）]

#### 角色 2：[角色名]
...

配置路径：多维表格右上角 → 高级权限图标 → 开启 → 添加角色
```
