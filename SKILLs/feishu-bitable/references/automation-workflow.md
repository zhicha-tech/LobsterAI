# 自动化与工作流参考

## 工作流 vs 自动化：如何选择

| 维度 | 自动化 | 工作流 |
|------|-------|-------|
| 定位 | 简单流程（2-4 步） | 复杂流程（多步+分支） |
| 界面 | 弹窗式，聚焦单步 | 画布式，可缩放拖拽 |
| 逻辑节点 | 不支持 | 支持条件分支/多分支/循环 |
| AI 节点 | AI 更新记录、自动化插件 | AI 生成文本/分类/Agent |
| 节点命名 | 不可自定义 | 可自定义 |
| 节点排序 | 不可拖拽 | 可拖拽 |
| 额度 | 共用每月自动化流程额度 | 共用（工作流占用自动化次数） |

**选择建议**：
- 监测变动 + 发通知 → **自动化**
- 状态变更 + 通知 + 记录时间 → **自动化**
- 审批流 / 多步骤流转 / 条件分支 → **工作流**
- 需要 AI 分类/Agent → **工作流**

**API 支持**：两者均不可通过 API 创建，搭建时输出配置建议供用户手动操作。

## 触发条件一览（8 种）

| 触发条件 | 说明 | 典型场景 |
|---------|------|---------|
| **添加新记录时** | 新增记录时触发。须至少选择一个字段"不为空"避免空记录触发。不支持按钮字段和人员扩展字段作为非空条件。 | 表单提交后通知负责人 |
| **修改记录时** | 指定字段内容变更时触发。可选"所有记录"或"指定记录"。不支持关注：按钮、自动编号、创建人、创建时间、人员扩展字段。 | 进展变更为"已完成"时记录时间 |
| **新增/修改的记录满足条件时** | 字段变更且满足特定条件时触发。已有记录满足条件不触发。与"修改记录时"区别：只有设置的字段本身变更才触发。 | 金额>10 万时通知主管 |
| **到达记录中的时间时** | 到达日期/创建时间/公式字段指定时间时触发（一次性）。 | 截止日期当天提醒 |
| **定时触发** | 在特定时间触发，支持重复。注意：截止日期当天不触发；31 日重复时无 31 日的月份跳过。 | 每天 9 点发日报 |
| **点击按钮时** | 点击数据表按钮字段、仪表盘按钮组件、消息卡片按钮时触发。阅读者也可触发。 | 阅读者点击按钮加入群组 |
| **接收到飞书消息时** | 群组/单聊中收到消息时触发。可设条件：关键词、指定用户、新话题、含图片/文件。后续可引用消息 ID/内容/发送人等。 | 群内关键词消息自动录入表格 |
| **接收到 webhook 时** | 外部系统发送 POST 请求触发。支持凭证校验（Bearer token）、IP 白名单、幂等（Client-Token）。 | 外部系统数据同步 |

## 执行操作一览

### 基础操作（所有流程可用）

| 操作 | 说明 | 关键限制 |
|------|------|---------|
| **发送飞书消息** | 发送卡片消息给个人/群组。可选身份：个性化应用/个人用户/助手/自定义机器人。支持附件和按钮。 | — |
| **发送飞书邮件** | 自定义收件人/主题/正文，可引用附件字段。 | 需管理员启用邮箱 |
| **新增记录** | 在数据表中新增一条记录，可设置字段值。 | 不支持设置：自动编号/查找引用/按钮/关联/创建人/修改人/创建时间/更新时间/需计算的公式/人员扩展字段 |
| **修改记录** | 修改全部或筛选条件匹配的记录。可同时修改多字段。 | 同上 |
| **查找记录** | 查找符合条件的记录用于后续步骤。不可作为最后一步。可引用：所有记录/第 1 条/某列值。 | 查找引用字段作为条件时仅支持等于/包含/为空 |
| **发送 HTTP 请求** | 调用第三方 API。 | — |
| **延迟** | 上一步执行后延迟指定时长再继续。 | 1 分钟 ~ 2 小时 |
| **发送 Outlook 邮件** | 向 Outlook 邮箱发送邮件。 | — |

### 逻辑操作（仅工作流）

| 操作 | 说明 |
|------|------|
| **条件判断** | if-else 分支 |
| **多分支** | 多个条件分支 |
| **循环** | 对数据集循环处理 |

### 飞书应用操作

| 操作 | 说明 | 注意事项 |
|------|------|---------|
| **创建飞书日程** | 设置日程主题/时间/参与人。默认提前 5 分钟提醒。 | 建议创建"日历 ID"/"日程 ID"文本字段记录 ID |
| **删除飞书日程** | 通过日历 ID + 日程 ID 删除。 | 需提前记录 ID |
| **创建飞书任务** | 设置任务标题/负责人/截止时间/关注人。 | 建议创建"任务 ID"字段记录 |
| **完成/重启/删除任务** | 通过任务 ID 操作。 | 需提前记录 ID |
| **创建飞书群组** | 最多 50 个群成员。 | — |
| **添加/移除群成员** | 选择群组和成员。 | — |

### AI 操作

| 操作 | 可用范围 | 说明 |
|------|---------|------|
| **AI 生成文本** | 自动化+工作流 | 配置指令，引用字段内容生成文本 |
| **AI 更新记录** | 仅自动化 | 更新 AI 字段（分类/标签/翻译/自定义/总结/信息提取） |
| **AI 分类** | 仅工作流 | 按预设规则分类，执行对应分支 |
| **AI Agent** | 仅工作流 | 智能规划，自动调用工具完成任务 |

### 插件/捷径

| 操作 | 可用范围 |
|------|---------|
| 自动化插件 | 仅自动化 |
| 节点捷径 | 仅工作流 |

## Webhook 规格

### 请求格式

```bash
curl --location --request POST '[webhook地址]' \
  --header 'Authorization: Bearer [token]' \
  --header 'Content-Type: application/json' \
  --data-raw '{"key": "value"}'
```

### Header 参数

| 参数 | 必填 | 说明 |
|------|------|------|
| Authorization | 否 | 凭证校验。格式：`Bearer [token]`（Bearer 和 token 之间有空格） |
| Client-Token | 否 | 幂等控制。传 uuid，相同值 3 小时内只触发一次 |

### 频率限制

| 范围 | 上限 |
|------|------|
| 整个多维表格的 webhook 触发 | 50 次/秒 |
| 单个自动化流程 | 5 次/秒 |
| 不开启凭证校验的流程 | 1 次/秒 |

### 请求体大小限制

4 MB

### 错误码

| 错误码 | 原因 | 解决方案 |
|--------|------|---------|
| 800005649 | 凭证不匹配 | 检查 Bearer token |
| 800005650 | IP 不在白名单 | 添加 IP 到白名单 |
| 800005647 | 请求体超 4MB | 减少内容 |
| 800005646 | HTTP 节点 URL 与 webhook 地址一致导致死循环 | 修改 HTTP 节点 URL |
| 800005652 | 触发频率超限 | 降低频率 |
| 800006003 | 请求体 JSON 格式错误 | 检查 JSON 格式 |

## 给用户的配置建议模板

搭建完成后，为用户输出自动化/工作流配置建议时使用此模板：

```
### 建议配置的自动化/工作流

#### 自动化 1：[名称]
- 触发条件：[选择触发类型] — [具体字段/条件]
- 执行操作：
  1. [操作1]：[具体配置]
  2. [操作2]：[具体配置]

#### 工作流 1：[名称]
- 触发条件：[选择触发类型]
- 流程节点：
  1. [触发] → 2. [条件判断] → 3a. [满足时操作] / 3b. [不满足时操作]
```
